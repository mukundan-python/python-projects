# Caesar Cipher Tool

A Python Tkinter GUI application for encrypting and decrypting text using the Caesar Cipher.

---

## Features

- Encrypt or decrypt text with a numerical key
- Simple and clean graphical interface
- Input validation for numeric keys
- Easy-to-use for beginners and advanced users

---

## How It Works

1. Enter your text in the input field.
2. Enter a numeric key (shift value).
3. Select "Encrypt" or "Decrypt".
4. Click "Process" to get the result.

The program shifts letters according to the key while leaving non-letter characters unchanged.

---

## Technologies Used

- Python 3
- Tkinter (built-in GUI library)

---

## How to Run

1. Install Python 3.x
2. Download or clone the repository
3. Run:

python caesar_cipher_gui.py
