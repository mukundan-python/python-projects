Number Base Converter
A GUI-based number base conversion tool built using Python and Tkinter.

This application allows users to convert values between:

Binary (Base 2)
Decimal (Base 10)
Hexadecimal (Base 16)
Overview
This project was created to demonstrate understanding of:

Number base systems
Python data type conversion
GUI development using Tkinter
Input validation and error handling
The program converts user input into decimal internally and then reformats it into the selected base.

Features
Clean graphical interface
Conversion between all three number systems
Error handling for invalid inputs
Structured layout using Tkinter
